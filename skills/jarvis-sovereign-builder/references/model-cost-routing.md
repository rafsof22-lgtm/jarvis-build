# Model and Cost Routing

Rank routes by quality, reliability, speed, privacy, reusability, context fit, modality, tool/coding capability, hardware fit, cost, latency, risk and lock-in.

Mandatory order: verified artifact/cache; deterministic local software; internal DB/API; local/private model; included capability; free allowance; cheapest qualified paid route; best-value scalable tier; premium specialist; multi-model panel only when value justifies it; qualified human for regulated/unresolved high-risk work.

Route classes: deterministic, local/private, economy cloud, specialist, parallel panel, human escalation.

Dedicated cost agents: Free-Tier Scout, Open-Source Scout, API Price Monitor, Hosting Optimiser, Token Optimiser, Cache/Dedup, Batch Execution, Subscription Consolidator, Vendor Lock-In Assessor, Cost Forecast, Quality-per-Dollar Evaluator and Paid-Escalation Approver.

Track task, candidates, selected route, model/provider/version, parameter class, price basis, tokens/calls, latency, quality, retries, fallback, cache hit, quotas/credits/CI minutes/compute/storage/bandwidth, expiry, migration and paid-escalation justification. Never silently create billable resources.
