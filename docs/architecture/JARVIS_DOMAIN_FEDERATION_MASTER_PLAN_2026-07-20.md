## 1. Executive decision

Jarvis will retain a detailed modular backend while presenting a small, novice-friendly owner interface.

The system must not delete or flatten specialist capabilities merely to make the interface simpler. Instead it will use progressive disclosure:

- one owner-facing page;
- seven primary workspaces;
- clear status, next action, cost and risk cards;
- specialist modules, agents, skills, integrations, records and advanced parameters behind each workspace;
- shared governance
