# Verified Jarvis Module Handoff Standard

Status: READY_FOR_REVIEW

## Scope

Apply this standard to the XRP/HBAR Framework Options module and to future independently usable Jarvis modules when their source material is accessible.

Do not restart, simplify, overwrite, compress away or remove approved requirements. Preserve raw sources, exact pointers, hashes, versions, amendments, superseded history and unresolved conflicts.

## Required source accounting

Create and maintain a known source denominator covering visible user requests, adopted assistant responses, uploaded/project files
