# Project Instruction Output Policy

## Canonical source

The Jarvis canonical project constitution and repository specifications are unlimited, additions-only artifacts. Their size must not block Skill packaging, CI, reconstruction, testing, deployment, tracking or release work.

## ChatGPT custom project instructions

A ChatGPT custom-project-instruction field is a separate deployment surface and may have a platform-enforced character limit. Generate a compressed paste-ready projection only when the user explicitly requests custom or project instructions.

When explicitly requested:

1. Start from the unlimited canonical constitution.
2. Preserve all applicable safety, approval, evidence, continuity, source, rollback and trigger requirements.
3. Fit the current platform limit.
4. Return one complete replacement block, exact character count and compact changelog.
5. Do not claim automatic installation without tool evidence.

During ordinary Jarvis work:

- do not print the full custom-project-instruction block;
- update the canonical constitution and related registries instead;
- do not treat an over-limit paste-ready projection as a Jarvis build blocker;
- classify the condition as `OUTPUT_ADAPTATION_REQUIRED`.
